## What Is Agent2Research?

Agent2Research is a medical research AI tool intelligence and decision guide. It helps researchers choose AI and research tools by task, not by hype.

It organizes tools and workflows around:

- Literature search and evidence discovery
- Systematic reviews and screening
- Zotero workflows and reference management
- Academic reading and writing
- Scientific visualization
- Medical NLP
- Bioinformatics
- Medical imaging
- Research productivity and tool comparison

## Why This Repository Exists

This repository is a public, shareable introduction to Agent2Research. It is designed for directory submissions, resource pages, GitHub awesome lists, academic communities, and people who want a quick multilingual overview before visiting the product.

## English

Agent2Research is built for medical researchers, clinicians, students, and research teams who need to evaluate AI tools in real work contexts.

Instead of listing tools as a generic directory, Agent2Research focuses on practical questions:

- Which tool is suitable for this research task?
- What should I try first?
- Where can this workflow break?
- How does the tool fit into literature review, systematic review, Zotero, writing, bioinformatics, or imaging workflows?

Start here: [Agent2Research](https://agent2research.com/)
