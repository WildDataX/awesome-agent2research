# Contributing

Thanks for helping improve this resource.

Good contributions include:

- Useful AI tools for medical research
- Systematic review workflow resources
- Zotero workflow resources
- Academic writing and literature review tools
- Bioinformatics and medical imaging tools
- Evaluation criteria for research tools
- Directory or community pages where Agent2Research may be useful

Please keep contributions:

- Specific
- Evidence-based
- Useful for researchers
- Free of spam or link exchange requests

Open an issue or pull request with a short explanation and relevant links.
